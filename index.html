La façon dont je travaille

Lazy Lunch

Mobile web application développer dans un cadre pedagogique.

L'application à pour but d'offrir les services proposé à la cafétéria de l'ESIAJ mias en ligne.

Elle permet donc de consulté le menu, vérifié le budget sur votre carte, commander, voter pour la suggestion de la semaine suivante, savoir quand aller faire la file.

Développé grâce à jQuery, Hammer.js

DW'M'Twitter

Datavisualisation développé dans un cadre pédagogique.

Le but étant de présenter les resultat d’une étude mener sur l’utilisation de twitter par les élèves de DWM de l’ESIAJ. Et de montrer l’impact de ce derniers sur leurs résultats de fin d’année.

Développé grâce à jQuery, D3.js, SVG, Waypoint.js, JSON

Ce plugin a pour but d'intégrer facilement un système de particule sur votre site. Et de pouvoir écrire avec ces particules.
