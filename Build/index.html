Lazy Lunch

Mobile web application développée dans un cadre pédagogique.

L'application a pour but d'offrir les services proposés à la cafétéria de l'ESIAJ mais en ligne.

Elle permet donc de consulter le menu, vérifier le budget sur votre carte, commander, voter pour la suggestion de la semaine suivante et de savoir quand aller faire la file.

Développée grâce à jQuery, Hammer.js

DW'M'Twitter

Datavisualisation développée dans un cadre pédagogique.

Le but étant de présenter les résultats d’une étude menée sur l’utilisation de twitter par les élèves de DWM de l’ESIAJ. Elle permet de montrer l’impact de ce dernier sur leurs résultats de fin d’année.

Développée grâce à jQuery, D3.js, SVG, Waypoint.js, JSON

backgroundify.js

Plugin jQuery développé pour un projet personnel.

Les éléments css n'ayant aucune sémantique, l'utilisation d'un background image fait perdre la potentielle semantique de cette image.

C'est pour répondre à ce problème que j'ai créé ce plugin qui va transformer vos images html en image background css.

Développé grâce à jQuery
